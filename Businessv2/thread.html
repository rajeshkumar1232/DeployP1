Thread design using lamda

  • The Runnable interface is a functional interface with a single method run(). By using a lambda expression, we define the run() method's body directly within the Thread constructor.
  • Thread.start() starts the thread and executes the code inside the lambda expression asynchronously.
        
			public class ThreadLambdaExample {
				public static void main(String[] args) {
					Thread thread = new Thread(() -> {
						System.out.println("Thread is running using a lambda expression");
					});
					thread.start();
				}
			}
