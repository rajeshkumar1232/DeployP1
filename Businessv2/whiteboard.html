Project Name: Whiteboard

This project, Whiteboard, is an interactive platform designed for teaching and presentations. The front-end is built using React (integrating Excalidraw.io), and the back-end is powered by Spring Boot.

Users/Roles:

1. Student/User

  • Can view scheduled classes.
  • Can participate in live classes and interact in real-time on the whiteboard.
  • Can view recorded classes (stored sessions).
  • Can track progress in courses or assignments.

2. Admin

  • Can schedule or cancel classes.
  • Can view ongoing classes (track live sessions).
  • Can view stored classes (previous class sessions).
  • Can enroll or deroll students in specific classes or courses.
  • Can monitor student progress (access reports and metrics).
  • Can assign teachers to specific students or groups.
  • Can take demo sessions of teachers (evaluate teaching quality).
  • Has access to a dashboard with system statistics and activity logs.

3. Teacher

  • Can create and manage interactive teaching sessions using Excalidraw (e.g., adding annotations, diagrams, etc.).
  • Can assign tasks, assignments, or materials to students.
  • Can view student performance and progress in their classes.
  • Can interact with students in live sessions using features like a virtual whiteboard, chat, and file sharing.

Technical Requirements:

Frontend (React + Excalidraw.io):

  • Use React to build the user interface.
  • Embed Excalidraw.io for interactive whiteboard features, enabling students and teachers to collaborate in real-time.
  • Support live streaming classes, using technologies like WebRTC or external tools (e.g., Zoom, Jitsi).
  • Implement role-based authentication and authorization using JWT or OAuth2.
  • Create a responsive dashboard for Admin and Teachers to manage classes and view progress.
  • Ensure a responsive design for access across devices (desktop, tablet, mobile).

Backend (Spring Boot):

  • User Management: Implement role-based access using Spring Security. Provide APIs for registration, login, and profile management for Student, Teacher, and Admin roles.
  • Class Scheduling: Admin APIs for creating, updating, and canceling classes. Provide APIs to fetch class details (scheduled, ongoing, and stored).
  • Class Interaction: Integrate WebSocket or STOMP for live interactive whiteboard collaboration.
  • Progress Tracking: Admin APIs to monitor and track student progress. Integrate analytics tools for performance monitoring.
  • Enrollment/Assignment: Admin APIs for enrolling and derolling students, and assigning teachers to students or classes.
  • Storage of Classes: Use databases like MySQL or PostgreSQL for storing class data. Use cloud storage services (e.g., AWS S3, Google Cloud Storage) for saving videos, diagrams, and notes.
  • Progress Reports: APIs to generate progress reports for students and teachers, based on metrics like attendance and assignment completion.

Additional Features (Optional):

  • Notifications System: Integration for notifying students of upcoming classes, canceled sessions, or assignments via email/SMS.
  • Assignment/Exam Management: Teachers can assign homework or tests and track submissions.
  • Integration with LMS: Optionally integrate with learning management systems like Moodle or Canvas.

Technology Stack:

  • Frontend: React, Excalidraw.io, WebRTC (for live streaming), TailwindCSS/Bootstrap (for styling)
  • Backend: Spring Boot, Spring Security, Spring Data JPA, WebSocket/STOMP
  • Database: MySQL/PostgreSQL
  • Cloud Services: AWS S3, Google Cloud Services
  • Authentication: OAuth2/JWT for secure user sessions
